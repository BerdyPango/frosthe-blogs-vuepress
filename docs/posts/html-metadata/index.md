## Metadata 种类

**Title Metadata**: 负责设定浏览器标签页上网页的标题。这对于搜索引擎优化（SEO）来说至关重要，因为它有助于搜索引擎理解该网页的内容。

```html
<title>Page Title</title>
```

**Description Metadata**: 这段元数据简要概述了网页内容，并且通常会出现在搜索引擎的结果中。

```html
<meta name="description" content="A brief description of the page content." />
```

**Keyword Metadata**: 这段元数据包含了与网页内容相关的关键词，有助于搜索引擎对页面进行索引。

```html
<meta name="keywords" content="keyword1, keyword2, keyword3" />
```

**Open Graph Metadata**: 这种元数据能够改善网页在社交媒体平台上分享时的呈现方式，提供诸如标题、描述和预览图片等信息。

```html
<meta property="og:title" content="Title Here" />
<meta property="og:description" content="Description Here" />
<meta property="og:image" content="image_url_here" />
```

**Favicon Metadata**: 这种元数据将网页图标（即一个小图标）与网页关联起来，该图标会显示在浏览器的地址栏或标签页中。

```html
<link rel="icon" href="path/to/favicon.ico" />
```
